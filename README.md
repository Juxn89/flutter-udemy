## Quick Start

<ol>
	<li>Install <b>Dart SDK</b> from <a href="https://dart.dev/get-dart">dart.dev/get-dart</a></li>
	<li>Clone this repository:<br><code>git clone https://github.com/your-username/flutter-udemy.git</code></li>
	<li>Navigate to the project folder:<br><code>cd flutter-udemy</code></li>
	<li>Run any example:<br><code>dart run abstracClass.dart</code></li>
</ol>

> <b>Note:</b> No dependencies or virtual environment required. All files are pure Dart scripts.
